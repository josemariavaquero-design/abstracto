# Generador de Prompts de Arte Abstracto (v1.9 - Pantone Pro)

Este es un generador de prompts (indicaciones) para crear arte abstracto, basado en un archivo HTML/CSS/JavaScript autocontenido.

El proyecto es una herramienta para artistas digitales que deseen explorar la abstracción geométrica y lírica, utilizando paletas de colores Pantone predefinidas.

## Características Principales
* Generación de prompts en inglés y español.
* Selección de 30 Paletas de Color Pantone.
* Configuración detallada de geometría, estilo, composición e iluminación.
* Funcionalidad para guardar prompts y referencias de imágenes.

## Estructura del Proyecto
* **`ABSTRACTO.html`**: El archivo principal de la aplicación.
* **`vercel.json`**: Configuración para que Vercel sirva el archivo HTML como raíz.
* **`apple-touch-icon.png`**, **`icon-192x192.png`**, **`icon-512x512.png`**: Iconos de la aplicación para dispositivos móviles.

## Despliegue
Desplegado como sitio estático en Vercel.
